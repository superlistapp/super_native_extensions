### Compiling locally

#### Prerequisites

`super_native_extensions` is using [melos](http://melos.invertase.dev):
```bash
flutter pub global activate melos
```

### Boostrapping

```bash
git clone https://github.com/superlistapp/super_native_extensions.git
cd super_native_extensions
melos bootstrap
```
